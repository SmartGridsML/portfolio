Tennis Match Prediction System

An end-to-end machine learning system built to predict tennis match outcomes. This project demonstrates my ability to design, deploy, and monitor production-ready ML services.

System Overview

  • Feature engineering pipeline (15+ performance metrics)
  • XGBoost model training and evaluation
  • FastAPI inference service
  • AWS Lambda deployment with CloudWatch monitoring

Key Technical Decisions

  • Chose XGBoost for tabular performance and interpretability
  • Separated training and inference codepaths
  • Designed stateless API for scalable deployment
